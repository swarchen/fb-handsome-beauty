### 暑假ＦＢ型男正妹夯度大ＰＫ 排行榜

每年暑假都會有一群神奇的年輕人玩出一些神奇的活動
像是 [暑假ＦＢ型男正妹夯度大ＰＫ](https://www.facebook.com/events/622729007882144/)

因此想透過這個機會學習一下怎麼使用facebook graph api去爬取活動頁面的讚數和留言數

並且將資料存在mongodb當中 [facebook event crawler](https://github.com/swarchen/facebook-event-crawler)

這個網站為MEAN(mongodb, express, AngularJS, Node.js)架構
